# DNA Sequence Analyzer

A Machine Learning based DNA Sequence Analyzer developed using Python and Flask.

## Features

- DNA sequence analysis
- Organism prediction
- Mutation detection
- Severity/Risk prediction
- Web-based GUI using Flask

## Technologies Used

- Python
- Flask
- Pandas
- NumPy
- Scikit-Learn
- HTML/CSS

## Project Structure

├── app.py
├── templates/
├── dataset/
├── results/
├── report/

## Installation

1. Clone the repository

git clone https://github.com/yourusername/DNA-Sequence-Analyzer.git

2. Install dependencies

pip install -r requirements.txt

3. Run the application

python app.py
